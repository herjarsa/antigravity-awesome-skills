# 🚀 Protocolo Maestro de Inicialización Antigravity (v4.5)

Este documento es el estándar de oro para convertir cualquier repositorio en una **Fábrica de Software Agéntica**. Cualquier agente de IA que opere en este ecosistema DEBE seguir este protocolo al detectar el comando "inicializar antigravity" o al iniciar un nuevo proyecto.

---

## 🏗️ Fase 1: Infraestructura de "Brazos Robóticos"

**Objetivo:** Establecer la conexión física con el conocimiento maestro.

1. **Instalación Maestra:** Ejecutar el instalador desde la fuente canónica:

    ```bash
    node "C:\Users\herna\Documents\GitHub\antigravity-awesome-skills\bin\install.js" "<project_path>"
    ```

2. **Sincronización de Espejo:** Crear la estructura de compatibilidad `.agent/*`:

    ```bash
    bash scripts/sync-agent-structure.sh
    ```

## 🏭 Fase 2: Activación de la SaaS Factory (Identidad)

**Objetivo:** Inyectar la filosofía de eficiencia extrema y el "Golden Path".

1. **Adoptar GEMINI.md:** Aplicar las reglas de la "Fábrica Tesla":
    - **El Proceso > El Producto:** Los comandos y PRPs son activos vitales.
    - **Auto-Blindaje:** Cada error se documenta para que NUNCA vuelva a ocurrir.
    - **Golden Path:** Stack obligatorio (Next.js 16, Supabase, Tailwind CSS 3.4/4, Zod, Zustand).
2. **Arquitectura Feature-First:** Organizar el código por dominios funcionales, no por tipos de archivo técnicos.

## 🛠️ Fase 3: Inyección de Base Skills (El Yunque)

**Objetivo:** Equipar al agente con las herramientas críticas de supervivencia y evolución.

Es obligatorio importar y activar las siguientes habilidades base:

- `concise-planning`: Obliga a un plan WHAT/WHY/HOW antes de cada cambio.
- `lint-and-validate`: Control de calidad automático en tiempo real.
- `kaizen`: Mindset de mejora continua y estandarización.
- `systematic-debugging`: Protocolo de caza de bugs.
- `git-pushing`: Protocolo de guardado seguro.
- **`skill-forge` (CRÍTICO):** El motor de creación de habilidades de `tripleyak`. Permite al agente crear nuevas herramientas si no encuentra una adecuada en el repositorio maestro (960+ skills).

## 🧠 Fase 4: Conexión de Superpoderes (Neural Link)

**Objetivo:** Activar los sentidos externos mediante MCP (Model Context Protocol).

1. **Setup MCP:** Configurar los servidores especificados en `.agents/rules/mcp-superpowers.md`:
    - **Web Search/Fetch:** Para investigación profunda.
    - **Supabase MCP:** Para control total de la base de datos y Auth.
    - **n8n MCP:** Para orquestación de flujos externos.
    - **Playwright MCP:** Para validación visual y testing real en navegador.
2. **Alineación de ADN:** Leer `STACK.md`, `README.md` y `package.json` para ajustar el tono y la lógica técnica al contexto exacto.
